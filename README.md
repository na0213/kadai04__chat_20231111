# チャットアプリ

## 紹介と使い方
- シンプルなチャットアプリです。
- 残したいコメントを残すことができます。
- ペットと話せる会話風チャットもできます。

## DEMO
（任意）デプロイしている場合はURLを記入

## 工夫した点
- チャットで残したいコメントを探しにくくならないよう、残したいコメントを保存できる機能を備えました。
- 指定のネガティブワードを入力すると、自動でポジティブワードに変換できるようにしました。
- chatGPTのAPIを使用し、自動で会話できるようにしました。
